Constructor.

new mad.Control( element, options )

Parameters

  1. el {HTMLElement | can.NodeList | CSSSelectorString}

    The element the control will be created on

  2. options {Object}

    Option values merged with the class defaults and set as this.options

Returns

{mad.Control}

A new instance of the constructor function extending mad.Control.

Options

options.id

All controllers instantiated by the app should have an identifier.

The identifier can be defined either with the optional parameters of the Controller or it can be defined by the id attribute of the HTML element for which the controller will be created for.

Note that if the identifier has been twice in the optional parameters and in the template, the one defined in the optional parameters will override the one defined on the HTML Element.
