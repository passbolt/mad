Insert an html content and position it.

static.create(refElement, position, content)

Insert an html content and position it.

Parameters

  1. refElement {jQuery}

    The reference element to position the content with.

  2. position {string}

    The position of the content to insert, regarding the reference element.

    Available values : inside_replace, replace_with, before, after, first, last.

  3. content {string}

    The content to insert. The content parameter has to be a valid html string.

Returns

{HTMLElement}

The inserted element
