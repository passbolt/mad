Position an element in absolute

view.position(options, options.coordinates, options.coordinates.x, options.coordinates.y, options.reference, options.reference.element, options.reference.my, option.reference.at)

Position an element in absolute

Parameters

  1. options {array}

    Array of options

  2. options.coordinates {array}

    Position the element functions of the given coordinates

  3. options.coordinates.x {integer}

    Position the element functions of the given x coordinates

  4. options.coordinates.y {integer}

    Position the element functions of the given y coordinates

  5. options.reference {array}

    Position the element functions of a reference element

  6. options.reference.element {HTMLElement}

    The reference element

  7. options.reference.my {array}

    As per Jquery position plugin, the target corner of my element ("top left" by instance)

  8. option.reference.at {array}

    As per Jquery position plugin, the target corner of the reference element ("bottom left" by instance)

Returns

{void}
