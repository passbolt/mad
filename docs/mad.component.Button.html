The Button component is a simple button implementation.

new components_api.Button(element, options)

The Button component is a simple button implementation.

Parameters

  1. element {HTMLElement}

    the element this instance operates on.

  2. options {Object}Optional

    option values for the component. These get added to this.options and merged with defaults static variable

    • value : value of the button
    • events : object that can contain
      • click : callback to be executed at click.
    • tag : html tag to use.

Returns

{mad.component.Button}
